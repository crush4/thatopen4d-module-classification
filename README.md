# @thatopen4d/module-classification

ThatOpen4D Stage-1-Plugin: **Klassifikation** — projektinterne
Klassen-Hierarchie mit optionaler bSDD-/Uniclass-Verknuepfung und Import
aus geladenen Modellen.

## Installation

```json
"@thatopen4d/module-classification": "github:crush4/thatopen4d-module-classification#v1.0.0"
```

## Panel-Features

- Baum-View der Twin-Klassen mit CRUD-Dialog
- bSDD-/Uniclass-Picker fuer externe Klassen-Codes
- Import aus geladenen IFC-Modellen (batch-Anlage aus IFC-Kategorien)

## Build

```sh
npm install && npm run build
# → dist/plugin.js (~30 kB, gzip ~7 kB)
```
